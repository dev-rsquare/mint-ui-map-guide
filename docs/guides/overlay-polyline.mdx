# 오버레이 - 폴리라인

MapPolylineWrapper 컴포넌트는 폴리라인을 지도의 API 를 이용해 그립니다.


```javascript
const paths = [
  new Position(37.4002343929478, 127.108339583181),
  new Position(37.4002330712894, 127.107806511177),
  new Position(37.4004906706981, 127.107806987924),
  new Position(37.4004919924535, 127.108339948814),
  new Position(37.4002343929478, 127.108339583181), // 마지막 path 를 시작점으로 하지 않으면 라인이 닫히지 않는다
];

export function PolylineWrapperMap() {
  return (
    <MintMap base={{ center: new Position(37.4004906706981, 127.107806987924), zoomLevel: 19 }}>
      <MapPolylineWrapper position={paths} />
    </MintMap>
  );
}
```

<PolylineWrapperMap />

MapPolylineWrapper 컴포넌트는 PolygonMarker 와 유사한 기능을 합니다. (mode = 'POLYLINE')

[차이점](./overlay-polygon#polygonmarker-vs-mappolygonwrapper)은 MapPolygonWrapper 에 기술된것과 동일합니다.

## 기본 기능

### position

폴리라인의 path 좌표를 지정합니다.

주의할 점은 라인으로 처리하기 때문에, Polygon 처럼 닫힌 도형을 나타내려면 마지막 path 를 시작점으로 해야합니다.

### visible

이 값을 false 로 설정하면 해당 폴리라인이 숨겨집니다.

### editable

이 값을 true 로 설정하면 도형을 수정할 수 있습니다. (Google only)

## 폴리라인 스타일

### lineColor

외곽선의 색상을 지정합니다.

### lineSize

외곽선의 폭을 지정합니다.

### lineOpacity

외곽선의 투명도를 지정합니다.
