# 서클 마커

서클 마커는 원형의 마커를 그리는데 사용됩니다.

간단하게 원의 중심 좌표와 반지름을 입력하면 됩니다.

<CircleMarkerMap />

## 기본 속성

### center

원의 중심점 좌표를 지정합니다.

### radius

원의 반지름을 지정합니다.

### radiusUnit

원의 반지름을 지정할때 값의 유형을 지정합니다. 'PIXEL' 과 'METER' 를 지정할 수 있으며 기본값은 PIXEL 입니다.

## 기타 속성

다른 속성들은 [폴리곤 마커](./marker-polygon) 과 특징이 같습니다. 폴리곤 마커와 다른 점은 서클 마커는 svg 내부에 circle 엘레먼트를 생성한다는 점 입니다.
