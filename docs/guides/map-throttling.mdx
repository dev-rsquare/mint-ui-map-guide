# Bounds 변경 쓰로틀링

BOUNDS_CHANGE 이벤트는 지도의 센터 이동시에 일어나지만 지도의 컨테이너가 리사이즈될 때도 일어납니다.
이때는 연속적으로 여러번 실행되기 때문에 불필요한 동작이 순간적으로 많이 실행될 수 있습니다.
이를 방지하기 위해 옵션이 제공됩니다.

## boundsChangeThrottlingTime

이 값을 지정하면 해당 시간내에는 여러번 BOUNDS_CHANGE 가 일어나더라도 한번만 실행됩니다.

예를들면 boundsChangeThrottlingTime 이 500 인 경우, 500ms 동안 여러번 BOUNDS_CHANGE 가 일어나도 한번만 이벤트가 발생합니다.

```javascript
<MintMap boundsChangeThrottlingTime={500}>
```
